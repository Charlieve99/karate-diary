# Bitácora de Karate

Aplicación web sencilla para registrar clases de karate con fecha, tema y descripción.

## Archivos principales
- `index.html`
- `styles.css`
- `script.js`
- `config.js`

## Publicación
### GitHub Pages
1. Crea un repositorio en GitHub.
2. Sube esta carpeta.
3. En Settings > Pages, selecciona la rama `main` y la carpeta raíz `/`.
4. Guarda y espera a que se publique.

### Netlify
1. Crea una cuenta en Netlify.
2. Arrastra esta carpeta al panel de sitios.
3. Netlify publicará la web automáticamente.

## Nota sobre la base de datos
Para usar la nube con Supabase, edita `config.js` con tu URL y anon key de Supabase y crea la tabla con el SQL de `supabase-schema.sql`.
